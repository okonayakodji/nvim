# okonayakodji's nvim setup

## Dependencies
- `xclip`

## How to install?
1. Install nvim.
2. Install dependencies
3. Run:
```bash
git clone https://github.com/okonayakodji/nvim ~/.config/nvim
```
4. Install [packer](https://github.com/wbthomason/packer.nvim).
5. Run nvim.
6. Write `:PackerInstall`
